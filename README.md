# Portfolio - Moses Jairaj

A personal portfolio website showcasing my skills, projects, and contact information. Built with HTML, CSS, and JavaScript, with a responsive design and interactive features.

## Features
- About Me section
- Projects showcase with descriptions
- Contact form integrated with Formspree
- Downloadable CV
- Responsive design for mobile and desktop
- Smooth scrolling navigation
- Hamburger menu for mobile


## Technologies Used
- HTML
- CSS
- JavaScript
- Font Awesome for icons
- Formspree for contact form

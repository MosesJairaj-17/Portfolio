My Projects

Project - RecoWorld

Description

RecoWorld is a centralized platform that allows users to discover and explore books and movies in one place. It provides search and genre-based browsing, displays detailed content information, and enables users to save favourites for personalization. The platform ensures smooth performance with a Flask–MySQL backend and delivers an interactive, responsive UI for a seamless user experience.

Scope

  • Combines books and movies on one platform.
  • Supports advanced search and genre filters.
  • Stores detailed data (title, author/director, genre, etc.) in MySQL
  • Allows users to save favourites and create collections

Applicability

  • For users to explore and organize books/movies easily.
  • Useful for students, libraries, and institutions as a digital catalogue.
  • Suitable for entertainment enthusiasts and academic projects.
  • Scalable for commercial use as a recommendation platform.

Technologies Used

  • Frontend: HTML, CSS, JavaScript
  • Backend: Python (Flask Framework)
  • Database: MySQL Workbench
  • Authentication: Firebase (Google Sign-In)
